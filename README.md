Estructura básica del proyecto:
1.- Página principal index.html
2.- Subcarpetas para cada otra página.
3.- Cada página tiene sus propias carpetas css, img, js y data.
4.- Los css y js de bootstrap solo se encuentran en la carpeta css de la página principal (index.html)
5.- Las carpetas vacías tienen un ERASE_ME.txt, se debe borrar cuando la carpeta no esté vacía.
